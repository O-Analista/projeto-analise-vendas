# 📊 Projeto de Análise de Vendas

## 🔹 Objetivo
Criar um banco de dados no SQL Server e desenvolver um dashboard no Power BI para análise de vendas.

## 🛠 Tecnologias Utilizadas
- SQL Server
- Power BI
- DAX

## 📂 Estrutura do Projeto
- Script SQL para criação do banco
- Dashboard Power BI com KPIs

## 📈 KPIs Desenvolvidos
- Total de Vendas
- Ticket Médio
- Crescimento Anual
- Ranking de Produtos
